Larch Provisions Website

Epicodus Code Review .NET

Larch Provisions is a catering company in Corbett, Oregon. This project is their website.

This application consists of:

  • Landing, about and contact page for the company
  • Ingredients database
  • Recipe database that consists of ingredients
  • Menu database that consists of recipes
  • A weekly menu that can be created from recipes in the database.

Installing

  • Clone this repository
  • Navigate to the folder and run the project in Visual Studio 2015

Technologies used

  • Visual Studio 2015 ASP.NET 5 MVC
  • Design Framework: Materialize, SASS
  • Languages: Javascript, C#

Run & Deploy

  • Run your app
  • Run your app on .NET Core
  • Run commands in your project.json
  • Publish to Microsoft Azure Web Apps
